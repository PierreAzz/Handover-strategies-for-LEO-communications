Abstract

LEO satellite constellations provide high-capacity global communications but are challenged by frequent handovers due to satellite mobility. This paper introduces Adaptive Candidate Pruning (ACP) and Weighted Time Threshold Optimization (WTT-O) to enhance a baseline Hungarian-Matching-based handover decision algorithm. ACP filters weak links, reducing unnecessary handovers, while WTT-O introduces temporal stability inspired by 4G LTE Time-To-Trigger (TTT), adapted for the high-bitrate and load-distribution requirements of LEO networks. Both techniques ensure global optimization across all users and satellites. Simulation results demonstrate up to 25% increase in utilizable bitrate, 60% fewer handovers, and 70% reduction in computation time compared to the baseline.

1. Introduction

LEO satellite networks are highly dynamic, causing frequent handovers. Traditional Maximum Weighted Matching (MWM) algorithms recompute assignments at every time slot, which increases signaling overhead. In 4G LTE, Time-To-Trigger (TTT) ensures handover stability by only executing handovers if signal conditions remain consistent over a duration. While stability is essential in terrestrial networks, in LEO systems the primary goal is high bitrate due to long propagation distances. Direct application of 4G TTT is suboptimal; instead, we adapt its concept to enable event-driven updates while preserving global optimization and load balancing across satellites.

2. System Model

Let U = {u₁,...,u_M} and S = {s₁,...,s_N} represent GateWays or users and satellites, respectively. At each time slot t, define a weight matrix W_t ∈ ℝ^{M×N}, where W_t(i,j) is the achievable bitrate of user i from satellite j.

maximize: max_{X_t ∈ {0,1}^{M×N}} Σ_{i=1}^{M} Σ_{j=1}^{N} W_t(i,j) X_t(i,j)
subject to: Σ_{j=1}^{N} X_t(i,j) ≤ 1, Σ_{i=1}^{M} X_t(i,j) ≤ 1

Here, X_t(i,j)=1 indicates that i is assigned to satellite j at time t. The optimization considers the entire GateWay-satellite matrix to ensure global optimization.

3. Adaptive Candidate Pruning (ACP)

ACP removes weak links below a threshold τ_{ACP} to reduce unnecessary handovers:

W'_t(i,j) = { W_t(i,j), if W_t(i,j) > τ_{ACP}; 0, otherwise }

This ensures the matching focuses on strong candidates, improving computational efficiency and reducing signaling overhead while still achieving global optimization.

3.1 Expected Bitrate with ACP

R_{ACP} = Σ_{i=1}^{M} Σ_{j=1}^{N} W'_t(i,j) X_t(i,j)

4. Weighted Time Threshold Optimization (WTT-O)

WTT-O adapts 4G LTE TTT for LEO by introducing a threshold T_{WTT}. The previous assignment is maintained if link variations are below this threshold:

ΔW_t(i,j) = | W_t(i,j) - W_{t-1}(i,j) |
X_t(i,j) = { X_{t-1}(i,j), if ΔW_t(i,j) < T_{WTT}; 1 if assigned by MWM, otherwise }

Only significant changes trigger recomputation of MWM, preserving global optimization and reducing unnecessary local updates.

5. Algorithmic Implementation

-- BASE Algorithm -- for each time slot t compute MWM on W_t assign users to satellites X_t globally end
-- BASE + ACP Algorithm -- for each time slot t prune W_t: W'_t(i,j) = 0 if W_t(i,j) < τ_{ACP} compute MWM on W'_t assign users to satellites X_t globally end
-- BASE + WTT-O Algorithm -- for each time slot t for each i,j compute ΔW_t(i,j) if ΔW_t(i,j) < T_{WTT}, keep previous assignment X_{t-1}(i,j) else compute MWM on W_t and assign X_t(i,j) globally end
-- BASE + ACP + WTT-O Algorithm -- for each time slot t prune W_t: W'_t(i,j) = 0 if W_t(i,j) < τ_{ACP} for each i,j compute ΔW'_t(i,j) if ΔW'_t(i,j) < T_{WTT}, keep previous assignment X_{t-1}(i,j) else compute MWM on W'_t and assign X_t(i,j) globally end

6. Simulation Setup

We simulate M=3 gateways and N=9 satellites over K=360 time slots. Weight matrices are generated with random variations and occasional drops to emulate fading conditions. Parameters: HM_margin=0.08, α_adjust=0.1, τ_{ACP}=0.15, T_{WTT}=3 slots. We compare total matching weight, handover count, runtime, and utilizable bitrate across all variants.

9. Discussion

ACP reduces unnecessary handovers by filtering weak links, while WTT-O preserves stability without compromising global optimization. BASE+ACP+WTT-O achieves the highest utilizable bitrate with minimal handovers. These enhancements adapt 4G LTE concepts to LEO networks by prioritizing global optimization and throughput rather than only stability, addressing the long distances and high load in LEO constellations.

10. Conclusion

We introduced ACP and WTT-O for LEO handover algorithms with Hungarian matching. The combination significantly improves bitrate utilization, reduces handovers, and lowers computational overhead. Global optimization ensures that all user-satellite pairings are considered collectively, which is essential for high-throughput LEO networks.
